Ark Desktop Application

Project Description

Ark is a delivery management application built during a Software Engineering module (CS2103) in the National University of Singapore.
This application's main target audience are delivery companies and online store owners, and aims to help them manage their deliveries. The user interacts with the Ark application via Command Line Interface, and it also has a Graphic User Interface created by JavaFX.

Project Details

  • Language: Java
  • Tools: JavaFX, Github
  • Duration: September - November 2017
  • Role: Programmer

Google Maps Browser

Ark includes a built-in google maps browser so that users can easily check up the delivery address location of any parcel simply by clicking on the specific parcel card, eliminating the need to open a seperate window for a map.
We decided to use the delivery address postal code for the search, instead of the name of the location in the case that more than one locations have the same name and a mix-up occurs.

Parcel Status

Ark makes prioritizing parcels near its due date more easily by colour coding parcel statuses. Parcel can have either of 3 statuses - Delivered (green), Delivering (yellow), Overdue (red), which are automatically updated if overdue.
Delivered parcels are also moved to a seperate tab so that users can focus on the more important undelivered parcels.
